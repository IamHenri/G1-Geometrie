Décomposition en série de Fourier¶

D√©composition d'un signal $T_0$ pr√©iodique sous la forme suivante : $$s(t) = A_0+\sum_{n=-\infty}^{\infty} c_n e^{2 j \pi n f_0 t}$$ Les √©l√©ments important de cette formule sont les $c_n$. Ils d√©terminent la fonction transform√©e.

def synthesize(cn_array,n_array,f0,t):
    s=np.zeros(len(t),dtype=np.complex)
    for indice in range(len(n_array)):
        cn = cn_array[indice]
        n = n_array[indice]
        s = s + cn*np.exp(2j*n*np.pi*f0*t)
    return s

Exemple de signal :¶

Soit $s(t)$ le signal de p√©riode $T_0$ :

$s(t) = A$ si $ 0 \leq t < T_0/2$
$s(t) = 0$ si $ T_0/2 < t \leq T_0/2$

Signal carr√© de p√©riode $T_0$ valant $A$ ou $0$.

Pour la transform√© de Fourier on doit calculer $c_0$ et les $c_n$.

$c_0 = A/2$

$c_n = -jA/\pi n$ pour $n$ impair et $0$ sinon.

def compute_coef(A,n_array):
    cn_array = np.zeros(len(n_array), dtype=np.complex)
    for indice in range(len(n_array)): 
        n = n_array[indice]
        if n == 0 : 
            cn_array[indice] = A/2
        else : cn_array[indice] = (-1j*A/(np.pi*n))*(n%2)
    return cn_array
